# memo

## 構想
- 脆弱性メモの登録は以下の情報を脆弱性ごとに登録します。
  - タイトル
  - 対象URL
  - 脆弱性の種類
  - 危険度
  - 発見日時
  - 概要
  - 再現手順
  - リクエスト・レスポンス
  - 影響
  - 対策方法
  - スクリーンショットの保存場所
  - 対応状況
- メモの一覧表示
  - 登録した脆弱性を一覧で表示します。
  - 危険度、脆弱性の種類、対応状況などで絞り込みできるようにします。
- 検索機能
  - タイトル、URL、脆弱性の種類、本文などから過去のメモを検索できるようにします。
